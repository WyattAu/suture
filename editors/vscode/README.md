# Suture VS Code Extension

Provides Suture version control integration for Visual Studio Code.

## Features

- **Syntax Highlighting**: Conflict markers (`<<<<<<<`, `|||||||`, `=======`, `>>>>>>>`) are highlighted
- **Blame**: Run `Suture: Blame File` to see per-line commit attribution
- **Log**: Run `Suture: Show Log` to view commit history in a webview panel
- **Status Bar**: Shows current branch name

## Requirements

- Suture CLI must be installed and in your PATH
- Configure the path in Settings > Suture > Path

## Installation

Copy this directory to `~/.vscode/extensions/suture-0.1.0` and reload VS Code.
