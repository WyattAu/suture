What is Suture?

Suture is a patch-based version control system that goes beyond line-level diffing. It uses semantic drivers to understand the structure of your files — JSON objects, YAML maps, SQL schemas, Word documents — and merges changes at the logical level. Two edits to different JSON keys? No conflict. Two people editing different slides in a PPTX? No conflict.

Patch DAG

Changes are modeled as patches in a directed acyclic graph. Two patches conflict only when their logical address sets overlap — not when they happen to touch nearby lines.

16 Format Drivers

Native support for JSON, YAML, TOML, CSV, XML, Markdown, DOCX, XLSX, PPTX, SQL, PDF, images, and OTIO. Files without a driver fall back to line-based merge.

Drop-in Workflow

Init, add, commit, branch, merge, rebase, push, pull — the commands you already know. Or use Suture as a Git merge driver in your existing repos.

Mount as a Filesystem

FUSE and WebDAV mounts let editors save files directly into a Suture repo. Every save creates a patch automatically.

Semantic Merge

Two people edit config.json — one changes host, the other changes port. Git sees overlapping lines. Suture sees non-overlapping keys.

git merge 
{
  <<<<<<< HEAD
  "host": "prod",
  "port": 8080
  =======
  "host": "staging"
  "port": 3000
  >>>>>>> staging
}
suture merge 
{
  "host": "staging",   ← theirs
  "port": 8080         ← ours
}

Quick Start

$ suture init
$ suture config user.name "Your Name"

$ echo '{"host": "localhost", "port": 3000}' > config.json
$ suture add .
$ suture commit "base config"

$ suture branch staging
$ suture checkout staging
$ echo '{"host": "staging", "port": 3000}' > config.json
$ suture add . && suture commit "point to staging"

$ suture checkout main
$ echo '{"host": "localhost", "port": 8080}' > config.json
$ suture add . && suture commit "change port"

$ suture merge staging
$ cat config.json
{"host": "staging", "port": 8080}   ← both changes, zero conflicts

Install

Cargo Binary Homebrew AUR 
# From crates.io (requires Rust 1.85+)
cargo install suture-cli

# Build from source
git clone https://github.com/WyattAu/suture.git
cd suture && cargo build --release --bin suture

# Homebrew (macOS / Linux)
brew install wyattau/tap/suture

# Arch Linux (AUR)
paru -S suture-git
